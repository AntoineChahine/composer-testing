 # 2.0-api-laravel-sdk

## Environment Requirements: 

- php^8.1

## Requirements: 

- Laravel Pint ^1.13
- phpunit ^9.6
- guzzlehttp/guzzle

## Installing Pint:

* composer require laravel/pint --dev

## Running Pint:

* From the terminal run this command:
  ./vendor/bin/pint

* Note that you can add it as action on file save from settings:
